heading: Git Submodules
title: Git Submodules
---


Flightcontrol will read a `.gitmodules` file at the root of your repo and automatically clone all [Git submodules](https://git-scm.com/book/en/v2/Git-Tools-Submodules) defined in it.

Private submodules are cloned if you’ve granted Flightcontrol’s GitHub app permission to access the submodule’s repository.
