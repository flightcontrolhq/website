import { Collapse } from '@components/ui/collapse.mdx'

| **Method** | **Path** |
| :------------ | :----------- |
| **POST** | `services/invalidateCloudfrontStatus` |

| **Headers** | |
| :------------ | :----------- |
| Authorization | `authorization: [API key]` |

| **Body** | | |
| :------------ | :----------- | :----------- |
| **Parameter** | **Type** | **Description** |
| `version` | **number** | This is the version ID of the API, currently at version 1. |
| `params` | **object**: `{ invalidationId: string; }` | Type:
`invalidationId` is the ID you should receive when you initially call the invalidation API. |

**Full Endpoint URL** `https://app.flightcontrol.dev/api/services/invalidateCloudfrontStatus`

#### Request Body Example
```json
{
  "version": 1,
	"params" : {
	  "invalidationId": "cl2kx7acm04003f6gobvidq2i"
	}
}
```
#### Response Example
```json
{
	"status": "SUCCESS",
}
```
