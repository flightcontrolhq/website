## Authorization

You'll need to get an API Key from your Flightcontrol account. See how to get it [here](/http-api/authorization/api-keys).


## Supported methods

| Method | Path |
| :------------ | :----------- |
| Cache invalidation  | **POST** [`services/invalidateCloudfrontStatus`](/http-api/cloudfront/cache-invalidation-api) |
| Cache status | **POST** [`services/invalidateCloudfrontStatus`](/http-api/cloudfront/cache-invalidation-status-api) |
